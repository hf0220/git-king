2. Git和Github 有什么联系 
	答：Git是本地仓库，Github是远程仓库

4. 请写出从创建一个空文件夹到将文件夹内容推送到远程仓库的步骤
	答：git add .
		git commit -m "提交日志"
		git push 缩略名 分支名
5. git diff 命令的作用是什么 
	答：查看变更内容
6. 直接将工作区的文件删除和使用git rm命令删除的区别是什么 
	答：手动删除需要先删除本地目录文件，然后把删除操作添加到暂存区，之后提交到本地版本库
		自动删除是自动删除目录文件并添加到暂存区，然后提交到本地版本库
7. git rm 和 git rm --cached 的区别是什么 
	答：git rm删除文件，git rm --cached是解除跟踪但不删除
8. git reset HEAD 和 git checkout -- 两个命令区别是什么 
	答：git reset HEAD 取消已经暂存的文件
		git checkout --取消对文件的修改
9. git reset HEAD 和 git reset --hard 两个命令的区别是什么 
	答：git reset HEAD 取消已经暂存的文件
		git reset --hard 回退到指定版本号
10. 如何为本地仓库添加远程仓库的地址 
	答：git remote add 缩略名 远程仓库地址
11. 当push代码到远程仓库发生冲突，该如何处理
	答：执行 git pull 缩略名 分支名称 命令，将远程仓库代码下载到本地仓库
		手动处理冲突代码
		处理完成后重新push代码到远程仓库
12. git中的分支有什么作用 
	答：方便修复，更新，测试，防止发生冲突
13. 使用MyBatis 框架的基本步骤是什么 
	答：创建Meven项目
		添加依赖
		创建主配置文件myBatis-config.xml
		创建mapper配置文件，并在主配置文件中加载
14. 写出你知道的 MyBatis框架中用到的设计模式 
	答：单例模式
15. Mapper配置文件中resultType 和parameterType参数作用是什么 
	答：resultType参数作用是定义返回值的类型
		parameterType参数作用是定义字段类型
16. 在什么情况下需要注册typeAliases 别名 
	答：类的完全限定名
17. MyBatis中事务的默认处理方式是什么 
	答：sqlSessionFactory.openSession() 需手动提交 sqlSession.commit()
		sqlSessionFactory.openSession(true) 自动提交
18. 一级缓存和二级缓存的区别 
	答：一级缓存默认开启
		一级缓存在同一个Sqlsession中是共享
		二级缓存默认关闭
		二级缓存需要将被缓存对像序列化，实现序列化接口
		二级缓存在同一个SqlSessionFactory生产的SqlSession之间是共享的
19. 如何开启二级缓存 
	答：在mapper.xml文件中添加<cache/>节点
		需要将被缓存对象序列化，属性也必须序列化，实现序列化接口
